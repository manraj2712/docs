## Quick Start Examples

<Tabs>
  <Tab title="Trade">
    Execute swaps and bridges across any supported chains:
    
    ```typescript
    import { DZapClient } from "@dzap/sdk";

    const dZap = DZapClient.getInstance();

    // Get quotes for trade
    const quotes = await dZap.getTradeQuotes({
      integratorId: "your-app",
      fromChain: 1, // Ethereum
      data: [{
        amount: "1000000000", // 1000 USDC
        srcToken: "0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48",
        destToken: "0x82aF49447D8a07e3bd95BD0d56f35241523fBab1",
        toChain: 42161, // Arbitrum
        slippage: 1,
      }],
      account: userAddress,
    });

    // Execute the trade
    const result = await dZap.trade({
      request: buildTradeRequest(quotes),
      signer: signer,
    });
    ```

  </Tab>
  
  <Tab title="Zapping">
    Complex DeFi operations in single transactions:
    
    ```typescript
    // Get zap quote for liquidity provision
    const zapQuote = await dZap.getZapQuote({
      srcChainId: 1,
      destChainId: 42161,
      srcToken: "0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48", // USDC
      destToken: "0x724dc807b04555b71ed48a6896b6F41593b8C637", // Aave USDC
      amount: "5000000000", // 5000 USDC
      account: userAddress,
      recipient: userAddress,
      slippage: 1,
    });

    // Execute zap
    const zapResult = await dZap.zap({
      request: zapRequest,
      signer: signer,
    });
    ```

  </Tab>
</Tabs>

## Getting Started

<Steps>
  <Step title="Install the SDK">
    ```bash
    npm install @dzap/sdk
    # or
    yarn add @dzap/sdk
    ```
  </Step>

  <Step title="Initialize DZap Client">
    ```typescript
    import { DZapClient } from "@dzap/sdk";
    
    const dZap = DZapClient.getInstance();
    ```
  </Step>

  <Step title="Execute Your First Operation">
    Start with getting token information or quotes:
    ```typescript
    const tokens = await dZap.getAllTokens(1); // Ethereum tokens
    ```
  </Step>

  <Step title="Build Your Application">
    Explore our comprehensive guides and examples to build sophisticated DeFi applications.
  </Step>
</Steps>
