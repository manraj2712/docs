## DZap SDK

A comprehensive TypeScript/JavaScript SDK for interacting with the DZap protocol, providing utilities for DeFi operations such as swaps, bridges, DCA, and Zaps. This SDK abstracts away the complexity of contract interactions, and price fetching, making it easy to build on top of DZap.

## What You Can Build

The DZap SDK enables you to build sophisticated DeFi applications:

- **DEX Aggregators**: Build trading interfaces with best-price execution
- **Portfolio Managers**: Create multi-chain portfolio management tools
- **DeFi Dashboards**: Develop comprehensive DeFi analytics platforms
- **Automated Trading Bots**: Implement algorithmic trading strategies
- **Cross-chain Applications**: Build seamless multi-chain user experiences

## Architecture

The SDK is built with modern TypeScript and provides:

- **Type Safety**: Full TypeScript support with comprehensive type definitions
- **Modular Design**: Use only the features you need
- **Price Integration**: Built-in price fetching with multiple data sources
- **Error Handling**: Robust error handling and retry mechanisms
- **Gas Optimization**: Multiple approval modes for optimal gas usage
